# Amazing-Bowling

#### isTrigger
- 체크가 되어있지 않으면 오브젝트가 튕겨나간다.
- 오브젝트의 물리적인 표면을 없애준다.

#### Particle System
- 2D 이미지를 랜덤하게 생성해 특수효과를 만들어주는것.

#### Instantiate()
- 원본 게임 오브젝트를 넣어주면 그것을 새롭게 복사해준다.
